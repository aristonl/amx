# AMX: Improving Matrix Audio Encoding and Surround Decoding with Rule-Based Algorithms and AI-Driven Techniques
This repository contains the source code of **AMX**, an AI-enhanced
matrix surround sound codec that can encode an existing surround
sound mix into stereo and transform it back into surround sound
for playback.

> [!NOTE]
> This project is in early development.
